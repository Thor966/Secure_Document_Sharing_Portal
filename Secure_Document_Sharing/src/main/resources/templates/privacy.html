Privacy Policy

Last Updated: January 2026

1. Information We Collect

SecureDocs collects limited personal information necessary to provide secure document sharing services. This includes account details such as name and email address, along with document metadata including file name, access type, expiry configuration, and permission status.

2. Usage & Security Logs

For security and auditing purposes, SecureDocs may log access timestamps, token validations, and related metadata to detect suspicious activity and ensure platform integrity.

3. How We Use Information

Information is used strictly for authentication, access control enforcement, token validation, expiry management, and overall system security.

4. Data Storage & Retention

Documents are stored securely on server infrastructure. Expired and revoked permissions are automatically invalidated to prevent unauthorized access.

5. Data Sharing

SecureDocs does not sell, rent, or distribute user data to third parties. Information is processed internally only for platform functionality.

6. Policy Updates

This Privacy Policy may be updated periodically. Changes will be reflected with a revised "Last Updated" date.
