What is SecureDocs?

SecureDocs is a secure document sharing portal designed to ensure that sensitive files are shared only with authorised users.

It combines OTP-based authentication, controlled permissions, and automatic expiry to minimise data leakage risks.

Key Objectives

  • Prevent unauthorised access
  • Secure authentication using OTP
  • Track document lifecycle
  • Enable time-bound sharing
